# VIP Deli & Tacos - Website

![VIP Deli & Tacos Hero](link-to-your-screenshot-here.png)

## 🌮 Overview
This is a modern, responsive single-page application (SPA) built for **VIP Deli & Tacos**, a highly-rated "hidden gem" restaurant in Sacramento, CA. The site is designed to capture the unique dual-nature of the business: a high-quality sandwich deli by day and a vibrant, chill taco spot by night. 

## ✨ Features
* **Mobile-First Design:** Fully responsive layout optimized for customers looking up the menu on their phones.
* **Dynamic Menu UI:** Clean, easy-to-read sections for Hot Sandwiches, Cold Sandwiches, Salads, and Tacos.
* **Vibe-Focused Aesthetics:** A dark-themed UI with neon accents (orange, yellow, and blue) that reflects their physical menu and late-night atmosphere.
* **Business Info Integration:** Sticky navigation with quick access to operating hours, location details, and a one-click "Call to Order" button.

## 🛠️ Tech Stack
* **Framework:** React / Next.js
* **Styling:** Tailwind CSS
* **Deployment:** Vercel (or your preferred host)
